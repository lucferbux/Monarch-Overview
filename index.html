Enabling RHOAI Features

Full modular architecture modules powering core RHOAI capabilities

G

GenAI Studio

Complete AI experimentation platform with full BFF implementation. Enables users to create projects, run experiments, track metrics with MLflow integration, and manage the complete generative AI development lifecycle within RHOAI.

M

MaaS (Models as a Service)

First full modular architecture module deployed to production with complete Go BFF. Provides model serving management, endpoint configuration, and scaling controls for deploying ML models as managed services.

External Integrations

Our modular architecture enables seamless third-party integration via proxy patterns

P

Perses

Integrated via proxy service architecture (no BFF). The observability package proxies to the data-science-perses Kubernetes service, rendering Perses dashboards natively within RHOAI for real-time metrics on model serving, GPU utilization, and inference latency.

M

MLflow

Integrated via iframe embedding and proxy endpoints (no BFF). The /mlflow proxy routes to the MLflow service, allowing users to track experiments, compare model runs, and manage ML lifecycle directly within the Dashboard.
